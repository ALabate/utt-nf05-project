\subsection{Detailed Description}
A list widget with details when you click on the title line with a triangle or an arrow that indicate current state. Each element is composed by to Q\+List\+Widget\+Item \+: title and content. The list is sorted by title alphabetical order

{\bfseries Warning} \+: As each element is indexed by the title, it has to be unique.
